# Fehler sind keine Ausnahme

**Talk for the .NET Usergroup Rhein/Ruhr in Ratingen, 29.04.2026**

## Open the slides

Switch to the `slides/` directory, then run:

```sh
npm run dev
```

## Run the C code

Switch to the `src/c/` directory, then run:

```sh
gcc main.c -o main && ./main
```

## Run the C# code

Make sure you have the latest .NET 11 preview installed. Then switch to the `src/dotnet/csharp` directory, then run:

```sh
dotnet run
```

## Run the Rust code

Switch to the `src/rust/` directory, then run:

```sh
cargo run
```
